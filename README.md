
# Groupomania

Social media project based on reddit 



## Features

- AUTH system
- CRUD user
- CRUD post
- CRUD comment
- like/unlike




## Installation

Install my-project with npm

```bash
  backend : 
  npm Install
  nodemon server

  frontend : 
  npm Install
  npm start
```
    
## Tech Stack

**Client:** React, Redux, Node-SASS

**Server:** Node, Express, mySQL
